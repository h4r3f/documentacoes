The splendiz dashboard

In this tutorial, we will see how to setup and use the splendiz dashboard that comes with the app. With this dashboard, you will be able to see all the current orders with the appropriate information for each order. 

You can see an example of the dashboard here.

Once again, you do not need any programing skills to setup the dashboard. So let's get busy !

creating firebase account

The dashboard is made with a backend library called Firebase. To make the app send data to your own dashboard, you will need to signup to firebase first.

 

1 - Go to the firebase website and sign up

2 - You will be redirected to your account. Now click on "create new app" and give it the name and url you want

3 - copy the url of your new app (the one ending with firenaseIO.com). and put it somewhere you can get to easily, we will need that url to setup the dashboard.

 

And that's it for firebase, your server is set up and ready to be used. Now we have to link up the app and the dashboard with the link you got in step 3.

linking the app

First, we will link the app to the firebase server. So open the app.js file located in splendiz_pack/Splendiz/www/js/app.js 

 

on line 10 of the app.js file, you have this : app.constant('firebaseRef','YOUR_FIREBASE_URL');

Now simply replace the text YOUR_FIREBASE_URL with the url of the firebase app you just created' And don't forget to leave the quotes (the '' ).

as an example : if your url was myawesomeapp.firebaseIO.com the line 10 in the file app.js should be : app.constant('firebaseRef','myawesomeapp.firebaseIO.com ');

 

Now your app is ready to send data to the dashboard.

linking the dashboard

Now open the file called orderService.js located in splendiz_pack/dashboard/app/shared.

In the same way you did for the app, link the url of your firebase app on line 3 :

orders.constant('firebaseRef','YOUR_FIREBASE_URL_HERE');

replace YOUR_FIREBASE_URL_HERE with the actual url of your firebase app (the same used for the app).

 

 

opening and using the dashboard

If you want to upload your dashboard on the internet, you could do it very easily. But we are only going to show you how to open the dashboard on your computer without puting anything on the internet. 

To open the dashboard, you are going to need a local server on your computer. There are plenty of servers you can use but we are going to use a google chrome plugin that does the job very well.

1 - If you do not have google chrome, install it here

2 - now open google chrome and go to the chrome web store to download this app, called web server for chrome.

3 - install the app

4 - In the url bar, type 'chrome://apps' and launch the app server server.

5 - now on the web server app, click on 'choose folder' and navigate to the dashboard folder located in splendiz_pack

6 - now tick the option called "Automatically show index.html"

7 - To open the dashboard, click on the url under web servers url 

 

and Voila ! Your dashboard is ready ! the next time you will be opening the dashboard, you won't have to repeat all the steps. All you have to do now when you want to use the dashboard is opening google chrome, typing "chrome://apps" in the url bar, opening the web server and opening the link from there.
